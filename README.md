# CANoe CAPL Support

VSCode 扩展，为 CANoe CAPL 语言提供语法高亮、悬停文档、代码补全、跳转定义、查找引用和参数检查。

## 功能

- **语法高亮**：CAPL 关键字、函数、类型、字符串、注释
- **悬停文档**：将光标放在函数上显示详细文档
- **代码补全**：输入函数名自动补全
- **跳转定义**：转到变量/函数定义处
- **查找引用**：查找所有引用位置
- **参数检查**：内置函数参数数量检查
- **类型检查**：类型兼容性检查

## 支持的文件

- `.can` - CAPL 程序文件
- `.cin` - CAPL 包含文件

## 支持的 CANoe 版本

19.5.44 (其他版本可能也兼容)

## 安装

1. 下载 `canoe-capl-support-1.0.3.vsix`
2. VSCode: `扩展` → `从 VSIX 安装`

## 使用

打开 `.can` 或 `.cin` 文件即可使用。

### 悬停文档

将光标放在函数名上查看文档：

```
Write("hello");  // 把光标放在 Write 上
```

### 代码补全

输入函数名前几个字符：

```
Wri  // 按 Ctrl+Space 触发补全
```

### 参数检查

调用内置函数时自动检查参数数量：

```
WriteEx();  // 会报告错误：需要 3 个参数
```
